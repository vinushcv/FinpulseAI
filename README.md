# FinPulse: SME Financial Health Assessment Platform

## Overview
FinPulse is an AI-powered financial health assessment tool for SMEs. It uses Google's Gemini to analyze financial statements and provide actionable insights.

## Features
- **Financial Health Twin**: Simulation of business scenarios.
- **AI Analysis**: Automated insights on cash flow and risks.
- **Privacy First**: Secure handling of financial data.

## Setup

### Backend
1. Navigate to `backend`.
2. `python -m venv venv`
3. `venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. `uvicorn main:app --reload`

### Frontend
1. Navigate to `frontend`.
2. `npm install`
3. `npm run dev`
